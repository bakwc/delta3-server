Detailed Description

Класс, в котором сосредоточена информация о клиенте

Так же здесь размещены различные методы, касающиеся обработки данных, приходящих по сети этому клиенту. Клиент может быть неавторизованным клиентом, обычным клиентом или администратором. Среди основных методов, которые расположены в классе Client - функция отправки данных этому клиенту, функция обработки входящих данных, набор функция для парсинга команд протокола различного типа.
